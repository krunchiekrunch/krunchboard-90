## Core Setup

* **Board:** Raspberry Pi Pico
* **Firmware:** KMK
* **Matrix:**

  * Rows: `GP22, GP19, GP18, GP17, GP16, GP11`
  * Columns: `GP12 → GP10`
* **Diode Orientation:** `COL2ROW`

## Modules

* **Macros** – custom key sequences
* **MouseKeys** – cursor + mouse control
* **MediaKeys** – volume/media shortcuts
* **EncoderHandler** – rotary encoder support
* **Layers** – multi-layer keymap
* **TapDance** – dual-function tap/hold keys

## Special Features

### Layers

* **Layer 0 (Base)** – standard keyboard layout
* **Layer 1 (Fn)** – mouse controls, system keys, macros

## Mouse Keys (Fn Layer)

  * Cursor movement
  * Left click
  * Right click
  * Scroll up/down

## Rotary Encoder

* **Default layer:**

  * Turn left → Volume Down
  * Turn right → Volume Up
* **Fn layer:**

  * Turn left → Brightness Down
  * Turn right → Brightness Up
